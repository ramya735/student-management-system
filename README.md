
# 🎓 Student Management System

A simple and efficient **Student Management System** built using **Node.js, Express, and MongoDB**.  
This project allows users to manage student records with full CRUD operations.

---

## 🚀 Features

- ➕ Add new student details  
- 📋 View all students  
- ❌ Delete student records  
- 📊 Store data in MongoDB  
- 🌐 User-friendly frontend interface  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Tools:** VS Code, Git, GitHub  

---

## 📁 Project Structure


student-management-system

---

## ⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/ramya735/student-management-system.git


2. Navigate to project folder:

cd student-management-system


3. Install dependencies:

npm install


4. Start the server:

node server.js


5. Open in browser:

http://localhost:3000


---

## 🧪 Usage

- Enter student details in the form  
- Click **Add Student**  
- View list of students  
- Delete students as needed
